# Jouss Patches

Custom Android app patches for the [Morphe](https://github.com/MorpheApp) patcher ecosystem.

## 📱 Patches

| Patch | App | What it does |
|-------|-----|-------------|
| **Morolingo Premium** | Morolingo (com.vivredarija) | Unlocks premium features, bypasses Pairip license check + RevenueCat entitlements |

## 🚀 Usage

1. Open the Morphe app
2. Go to patch sources / add repository
3. Enter: `https://github.com/Joussflls10/Jouss-Patches`
4. Select your APK/XAPK and apply the patch

## 🔧 Building locally

```bash
cd patches
./gradlew buildAndroid
```

Output: `patches/patches/build/libs/patches-*.mpp`

## 📦 Release

The latest MPP is always available from the [releases page](https://github.com/Joussflls10/Jouss-Patches/releases).
